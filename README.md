## Setup & Installation

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
git clone <repository-url>
cd frontend-RicardoReisConsultoria
npm install
```

### Running Locally

```bash
npm run dev
```

Application runs on `http://localhost:5173` (or specified port)

### Build for Production

```bash
npm run build
```
